# G-Shock Watch

A premium, interactive landing page for G-Shock watches featuring advanced scroll animations and a high-tech aesthetic.

## Features

- **Dynamic Scroll Animations**: Elements reveal and animate as you scroll.
- **Interactive UI**: Custom sliders, hover effects, and 3D card flips.
- **Responsive Design**: optimized for desktop and mobile.
- **Tech Stack**: HTML, CSS, JavaScript (Vite).

## Setup & Run

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```
